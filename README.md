# django_blog

## Startup
- Install the repository to your local machine
- Make sure you have Python installed (3.8.3 preferred)
- Once you are in the root dir, run `cd project`
- `pip install pipenv`
- `pipenv shell`
- `py manage.py runserver`
- View at `http://127.0.0.1:8000/`
- Enjoy!
